# 何小鹏：从AI缝合怪到物理AI的豪赌——第二次访谈实录

## 开场：从血海游泳到押注未来

大家好，我是小俊。这集是小鹏汽车董事长兼CEO何小鹏的返场。在我们之前那次访谈（商业访谈录第70集）中，何小鹏形容造车就像在血海里游泳。在过去一年里，除了汽车这场血海战役，小鹏汽车也要面临AI和机器人这场面向未来的战争，后者的战线可能更漫长，造人形机器人的胜率也要比造车低得多。对此，何小鹏在2025年下了更大的赌注。我们这次从“赌”开始聊起，他也聊了人形机器人、AI的诞生与意外、技术变革下CEO的心路、新车G9等话题。不过对于正在发生的事情，他有一些顾忌，也有一些尚且不能说的秘密。接下来就是我对何小鹏的访谈。期待2026年，我们和AI共同进步。

何小鹏：物理世界的CEO不敢赌，我可能胆子比较大。我们做了一个巨大的赌注——我们把以前的那套体系就停下来了。那套体系花了小几十个亿。

**问**：直接来讲，何小鹏 Skill 的一个缺点会是什么？

何小鹏：我觉得每个人 Skill 了之后，都会发现这个人实际上是有非常多的缺点的。

**问**：机器人对于小鹏来说意味着什么？

何小鹏：我觉得这个世界上可能有非常多的解法。我们选了一个最像人的人形机器人。

**问**：你觉得胜率有多大？

何小鹏：大概有个两成。

**问**：在这样一个技术剧烈变革的时代，你焦虑吗？

何小鹏：当然焦虑，因为你会发现你以前的很多逻辑论，甚至你逻辑论的底层的定理、范式，甚至到公理上，你的价值观、人生观、世界观，在这些东西的变化下不是那么 work。

**问**：你今天从血海里游出来没有？

何小鹏：我觉得都在游，我也不觉得有血出来了现在。

## 第一部分：关于AI的使用与Token

### AI产品的使用态度

**问**：在节目正式开始之前，我想先对于你最近对于AI的体感来做一个小的调研。你最近使用最多的几个AI产品是什么？

何小鹏：那我还真不多，还是在非常传统的AI产品——全问豆包。但是在 Coding 方面，我们现在团队内部用得非常多，但是我个人不愿意使用。

**问**：为什么？

何小鹏：他们就跟我问：“哎，小鹏你自己要去用。”我举了一个非常有意思的例子：我们当年在做互联网产品的时候，如果你天天用产品，你很快会到细节去了。比如说你会觉得 Skill 是哪里不好用。如果你了解它，你会去思考，认为它的优点和量还会非常长、非常多。但你带用进去，你会看到它的缺点跟问题，然后你会聚焦怎么去解决（比如多人协作的 Coding 等问题），反而不容易像远方看。所以我对于科技快速变化的很多产品跟能力，我自己认为要用，但不要太深度地用，特别是做一号位。但是基层的，或者说有一些产品线跟它强相关的，你就要鼓励——什么方式都可以上，然后最后要看结果，然后慢慢地把它归一。这样才是比较好的。

### AI Coding 对企业的意义

**问**：AI Coding 对于一个企业的一号位来说有任何的意义吗？你觉得它会对小鹏也好，汽车产业也好，智能驾驶行业也好，带来什么变化？

何小鹏：我觉得它是对初级程序员的一个非常好的（目前来说还是一个辅助工具）。但是也许两年到三年后，会逼迫初级程序员都上到高级程序员。但是我觉得对于像智能辅助驾驶，或者像其他的强AI的能力，它的帮助是比较小的，它只是其中的一个工具。真正要把整个研发做好、整个体系建好，实际上本身来说，你可以认为它是在最应用层来帮大家的，那是AI Coding。但如果你在内核层，比如说你想写个操作系统，我认为最核心还是整个的研发，而不是在 Coding。

### Token的使用：数字世界 vs 物理世界

**问**：你现在每个月会使用多少 Token？公司会使用多少 Token？

何小鹏：我们不太看这个事情。当然有好多人过去一年都在提。我认为数字化的中小型公司提这个事情会非常多，应该特别重要；数字化的中大型公司要适度关注，不应该太过于全面，因为有些公司的数字化的支持的业务可能不一定是数字化的，可能是非数字化的。一个非常有意思的点：在非数字化的公司对 Token 需要多少？你看我们的车，我们之前做了一个有意思的小统计，我们七代的 VOA 一天只用三四个小时的话，会用多少 Token？我现在准确数字已经不太记得了，当然那是内循环了。换一个角度，实际上也就是，在数字AI能用AI使用的 Token 数量，远远低于在物理世界AI自己需要用的 Token 数量。但是这个没有意义的，因为它不是用人，用AI来满足它的需求。自动驾驶的汽车你可以认为是一台自动的 Machine，它用Token来用多少。所以我在这两个领域里面，认为是两个不同的角度。在物理世界，我认为Token的使用将来会是 Machine 自己使用多少 Token，产生多少对 Machine 和人的价值；而人用多少 Token，产生对人和企业的价值，是另外一个维度。所以它的指标应该是不一样的，它的KPI指标。

**问**：你觉得多少是一个合理的数量？

何小鹏：我不知道。我在内部尽量不给大家控制Token。很多人问，不控制Token一个季度就把一年的费用全 Cost Down 了。我觉得，如果他真的能做到，那么我最重要的是管理最异常的 Top 10 的十种情况。其他的我都觉得可以开放，因为你不确认一个月一个人花1000块人民币还是1万块人民币是最有价值的。而我们每个人的每个月的工资很有可能远远超过这个数。如果它真的有能力花更多的钱、产生更大的价值，你为什么你要限制它？

**问**：现在公司的Token分配主要在哪里？

何小鹏：我觉得主要还是 G9。上次我们的自动驾驶跟座舱团队合并了，他们团队又大。Token 是对企业内部员工的术语。我们某种角度，你可以说用了多少张 H100。一张 H100 我不知道一年能够用多少 Token？我还没有把这个换算比例。我们算的是，比如我给你3万张 H100 或者5万张，你使用什么业务能够高效率和高效能地使用。在这个里面我们来看这个算力的使用情况。我不太使用 Token 的使用情况。

**问**：你刚才提到异常值，有什么样的异常值？

何小鹏：我举个例子，我们最近专门对于 Data 做了一个整个的控制。很多人都说数据的价值，我觉得极少的公司现在看到数据的巨大成本，因为在数字AI领域里面的数据量很小，几十个 TB 就可以来训练了。在我们训一次数据，几十个 TB 到几百个 TB。所以对数据的管理、使用、存储，是一个巨大的钱。可能我们一年在数据上投入的直接的刚性成本都接近10亿以上。哪些数据是有价值的、哪些是临时有价值的、哪些是需要非常快速使用的、哪些是可以 Warm Up 再使用的——每一个使用的钱的数量都是千万级的。你可以进行分析，然后进行优化。所以巨大的可提高效率、可降低成本和提高效能的逻辑。实际上算力也是一样，我们对数据、对算力都是不同的 Team 专门去管这个事情。但是我们不太 Care。我看到有很多公司说是公司的工程师使用Token的数量，我觉得那个数量跟 Machine 使用的Token数量，和物理世界AI模型训练时需要的Token数量，那都是非常小的数字。

## 第二部分：何小鹏 Skill、AI含量与人才观念

### “何小鹏 Skill”的构想

**问**：最近 Skill 非常出圈，如果把你训练成一个“何小鹏 Skill”，你觉得应该给他提供哪些训练的数据？那个 Skill 会是什么样的？

何小鹏：我觉得数字的模型跟物理的模型今天还是让基础的白领和基础的蓝领比较容易 Skill 化。如果把我都能够现在有清晰的逻辑论去 Skill 化，意味着不光基础蓝白领，更高端的蓝白领将都会有巨大的风险。

**问**：CEO可以被替代？

何小鹏：也许过数十年，或者过100年，类似我的能力的确可以被 Skill 化，但是那个时候可能我们每个CEO也都会有更强的、更综合的能力。他可能丢掉了ABC，但是获得了新的DEF。因为你这个问题上次我们不光是想过，我们是深度思考过的。因为我们在做很多机器人的事情。换个角度，我觉得在这个里面，它有两个非常重要的矛盾：对于大部分的人会觉得我如何把我（或者它）的能力 Skill 化；但是对于一个模型角度来看，我怎么知道你 Skill 化的这个能力是一个“对”的能力，并且不断地 OL（例如在物理的仿生模型里面可以去强化）。实际上在这个里面，它跟 Coding 或者跟自动驾驶都是不一样的——Coding 跟自动驾驶相对来说是比较清晰的，我知道什么是错的。但是把何小鹏 Skill 化，你很难判断这个 Skill 化怎么是对的或者错的，是非常困难的。所以在这个体系里面，就是我们在构建体系，而不是在使用别人构建好的体系。从我的角度来看，这是完全两个不同的角度。

**问**：直接来讲，何小鹏 Skill 的一个缺点会是什么？缺陷？

何小鹏：我觉得每个人 Skill 了之后都会发现这个人实际上是有非常多的缺点的。你的呢？当然都有。

### 企业一号位的AI含量

**问**：作为一个CEO，你觉得你今天的AI的含量够吗？

何小鹏：我不知道。我觉得在数字世界的很多公司里面，它可能可以把AI的含量放到小百分之几十。我觉得在物理世界不应该。小鹏是一家数万人的企业，我觉得AI上面花15%到20%。而且我讲的AI是一个泛AI，包括了我们的自动驾驶，包括了我们的机器人，我们觉得在这块都算是AI的整个范畴。我觉得这是足够了。

**问**：人工智能企业如果在一边，制造企业在另一边，你的天平应该是怎样的摇摆？你在里面会有纠结和拉扯吗？到底选择做一个AI企业还是做一个制造业汽车企业？

何小鹏：我不这样理解，抱歉。我把汽车认为有几种研发：第一个是硬件的研发，第二个是软件的研发，而AI只是其中的之一（软件之一）。还有制造研发，你刚说到的。硬件的研发跟制造是两种不同的能力——你有能力设计出一个很漂亮的桌子跟椅子，但是你可能无法制造出来。所以我觉得一个汽车的企业里面实际上在这四种能力的研发是一个基础，甚至将来可能到第五种。这是我的看法。

### 人才观念：通才 vs 专家

**问**：你选了这个机器人的负责人，他后来对你的选人标准有过任何影响吗？你觉得这是一个可以沿用的方法吗？马斯克特别喜欢用通才型的人，不喜欢专家型。

何小鹏：我觉得可能在特斯拉的硅谷，更能找到更多可选择的通才型人。在绝大部分的国家、城市或者行业是很难的。我想说这是第一个。我觉得 LC，他非常注重他的说法叫“人才的密度”，我认为叫“人才的潜力”。他基本上都用最好的人。比如去年的年底到今年的上半年，我们大概招了接近80个博士毕业生（就光一个部门），他们都很贵，但是我们都愿意。你要有能力聚拢这样一帮人，也要愿意支持他们长期的探索。你要相信这群年轻的小伙子或者女孩，他们能够创造奇迹。我认为叫“人才的潜力”，他认为叫“人才的密度”。我觉得要用超级聪明的人去做超级困难的事情，而不是用非常清晰的方向、流程或工具去锻炼。我觉得在某些阶段就是非常对的。

## 第三部分：从AI汽车企业到物理AI企业

### 改名的背后：从“智能电动汽车”到“物理AI”

**问**：去年你们还在说你们是一个AI汽车企业。今年小鹏汽车刚刚完成了改名，叫做小鹏集团，你们说你们是一个物理AI企业。这个名字的转换背后的原因是什么？从一个AI汽车企业到一个物理AI企业，它的变化是什么？

何小鹏：我觉得这个变化刚刚开始，还有很多都不能清晰地描述。但是我可以来说一下。在过去的10年创业生涯中，实际上小鹏汽车都在做核心三个词、六个字：智能电动汽车。所以过去10年我们做出第一款车，量产，卖出第一个10万台，做出第二款车……进入这样的循环。实际上在2014年小鹏汽车创业的时候，基本上没有人相信智能化，但也有少数人相信电动化，更多人相信汽车是个大生意。在2025年，所有人都相信电动化是未来，因为大家看到中国和全球在新能源上的变化。在过去的数年里，智能化虽然发展得非常快，在汽车领域实际上也发展得非常不尽人意。因为从更早年就有很多公司（从丰田到Google，中国的百度，海外的特斯拉，中国的小鹏等）都在自动辅助驾驶做了很多事情。某种角度，我觉得它做到的效果没有足够的高度。那个时候我们仍然还是AI的算法加上软件的规则的能力的组合，我把它戏称为 **“缝合怪”**。实际上不是用整个AI驱动来设计汽车，也不是用整个AI的模型能力来构建这个体系——还是以软件为主的逻辑。

### 自动驾驶的技术路线：从“缝合怪”到真正的AI驱动

小鹏在去年（2025年）碰到了一个非常大的变化。我们当时同事在做两代新的自动辅助驾驶——VOAE的第一代和第二代。VOAE的第一代是在过去的端到端领域里把它放大，把模型放大，把软件的规则降低，把使用场景跟范围变强，增强了后段能力（例如强化、后训练）。这是一种方向。另一种方向是，我如何抛弃掉原来那个端到端的逻辑，是用一个更大的 **Foundation Model**，更多地思考如何把自动驾驶的上限先打开，再去收敛下限（少错误，按照约定和期待去行动、去运动）。但实际上它的泛化能力有很多地方做得很差。举例，今天就没有一家自动驾驶公司的软件能够在地下的停车场很流畅地开，所有的停车场开的都是叫“记忆辅助驾驶”——它开了一次之后知道你的停车位和大概行车路线，对于整个物理世界的了解度非常低。核心还是不够智能，它的上限太低了。你可以认为要打开到1万分的可能性才能做好，但他们可能都在1000分左右（一个假设）。

所以去年差不多这个时候，我们的另外一代 VOAE 让我打开了一个新的变化。我认为它的上限可能可以到10万分到100万分，但是那个时候的下限也很惨烈。本来你希望做一个产品，上限1000分、下限900分，但那时候下限可能只有100分。所以它比其他的、我们自己的产品的下限上要更低，它的工程问题有很多。但是我们最后下了一个非常巨大的赌注——我们以前使用软件的工程、软件的流程，加上AI的算法和AI的工具（包括刚才提到的 Coding），你可以认为它是在一个业务流，还是软件的，在部分重要的工具环节和流程环节使用AI。但我们认为它是错误的。

**问**：这还是“缝合”？

何小鹏：是。它做出来的软件还是软件，因为你是用软件的方法论、使用AI的工具箱，做出来的是一个更强力的软件。我认为它叫做 **AI缝合怪**。所以我们去年做了巨大的转型和赌博，我们把以前那套体系就停下来了。那套体系花了小几十个亿。

**问**：去年的什么时间？

何小鹏：大概就是现在（当前访谈时间点）。

**问**：原因是什么？

何小鹏：我认为它无法做到无人驾驶，也不可能让机器人真正泛化。比如今天我们到了一个陌生的场所，机器人可以走过来说“小俊你好”，然后坐下来，你说“我就不给你递杯水了”，它会说“对，我不需要”。我觉得它可能是一个强规则加少数算法使用AI的能力。那一刻我们判断，智能电动汽车实际上我们所期待的“非常聪明的汽车”，但用的方法永远不会无限聪明。

### 为什么去年才意识到？

**问**：小鹏已经创业了11年，你们做智能化也非常早。

何小鹏：我觉得绝大部分人不是站在 **物理AI** 的角度去看一个事情的。我对于数字AI能改变很多东西，和物理AI能改变很多东西，是完全两种不同的方法论和路径。但绝大部分人还是以科研的角度去看数字AI。数字AI实际上某种角度是用人类的 Language，物理AI不是用人类的 Language。数字AI是人类语言被高度概括和浓缩的（语言即世界）。但是在物理世界，我们每个人每一天看到的数量根本无法用语言描述、还原和复制，它太大了。以前汽车领域的CEO很容易用数字世界的AI，了解了之后觉得“我好像懂它逻辑了”，去映射、扩展和使用。但那个时候没有任何一个CEO会说“我全部都要用”，因为不知道它到底有多少有效。因为在数字AI的市场里，很多模型就是跑个分、比比比分。我觉得在物理AI那是可笑的，因为它不光要比上限（跑分），还要比下限，还要保住下限，还要拼广度——品质、成本、材质、细节、法规允许度，这全是短板、长板和窄板。窄板要拓宽，短板要拓长，长板要做得更长。在数字世界核心看的是长板，不太看另外两块板。所以物理世界的CEO要不然不敢赌，要不然觉得“我还有好多块板，我怎么办”。我觉得我可能胆子比较大。我在那个时候也不知道它能不能补短板、能不能让窄板变宽。我们做了一个巨大的赌注——我们觉得，嗯，相信物理AI，换一种思路、换一种流程、底层要换一种组织方式，有可能能够做到不一样的点。所以我们走的道路跟今天看到的很多大模型公司，或者在模型上做应用的公司，都是完全不同的道路。这也是为什么我觉得 **Token** 都是应用型公司所Care的，我们早早都不是这个维度。

因为在以前我做智能电动汽车的时候，总觉得有一个很痛苦的点：无论怎么看，硬件占一个客户所需的比例都远大过50%。我认为在下一个十年，很有可能硬件和软件可能可以各占到50%。所以要用AI的方式重新驱动你的组织，设计你的软硬件的合体，并且获得更大的AI的价值。只有这样在新的十年里，软件的综合表现力（用户愿意为它付费的价值）才会占到一辆车或一个机器人的50%。我觉得这是一个非常大的改变。这也是为什么小鹏开始思考如何把 **Physical**（含汽车，也不仅限于汽车）和 **AI** 能够巨大地融合。我们的AI，对于数字AI我觉得基本上都是使用——比如Cloud Code非常好，我会用；如果Deep Seek Code在某个方面也非常好且成本也很棒，我就把那一部分切换过去。我们做了一个Hub，可以很容易地切换。但是 **Physical AI 的 Model 是不一样的，Data 是不一样的，Infra 是不一样的**。它最后如果能够把刚才讲的三块板都拼好，有可能做出不一样。这就是我们大概能说的框架吧，再多就不说了。

### 去年三四月发生了什么？

**问**：去年4月到底发生了什么？为什么就去年4月下了一个这么大的赌注？受了什么刺激吗？

何小鹏：没有，实际上就是体验不同的能力了。去年3月份我当时就在思考，为什么所有的公司都会觉得用更简单的方法论和更快的效能就能做出一个好像凑合可用的自动驾驶——它是对的。但是当它做了一年两年之后，它就会发现为了解决很多短板，它限制了自己的长板。所以它发现永远做不到 Level 4 或 Level 5。这是当时在去年年初的一个想法：这条路可能是一条捷径，但不是一条大道，是一条小路。我们就是要找到一条大道，真正能做到也许刚才所说的50:50的硬件加软件的 Value（不是技术人员思考的，是客户愿意为此买单的——用户愿不愿意买一个30万的车，其中15万算硬件，15万算软件的综合能力？有没有这样的车？现在没有。但我认为一定在这个十年会变化）。所以像物理的变化和AI的变化，重新思考，我觉得是一个非常重要的点。这也是我们小鹏在新的十年里大家会看到的。智能电动汽车可能会继续提，继续是我们的内核，但我们已经不是用电动或者汽车的方式去考虑智能，而是换一种逻辑。所有我们思考的企业的从战略的规划到研发的过程，都会逐步来进行调整。

**问**：去年三四月份有哪个标志性的会议吗？在公司内部，你下了这个非常大的赌注。去年上半年应该属于小鹏的销量回暖的时候，然后开始更多思考下一个十年的方向。

何小鹏：我觉得没有一个会议，基本上在脑袋里拍。全力以赴大概在去年的三季度末。实际上三季度末我们做了一个非常大的动作——我们把整个自动驾驶中心核心的组织架构全改了。每一个时间窗口期都有非常多优秀的同学，但每个人都有原来的惯性。你会习惯用过去的方法、想用最新的工具和技术去做出更好的东西，大部分时候就是对的。但有很多时候你的工作方法论都会改。所以调整方法论、调整 Mindset。

**问**：这听起来是你脑子里做的决策。在公司内部可能没有一个非常标志性的会议或节点。但你在脑子里形成这个决策之后，紧接着你做了什么呢？就是所谓的AI驱动的组织、用AI来做设计软硬件一体化，紧接着要做什么？对于这么大的一个组织来说，你能不能把你的步骤分享一下？

何小鹏：对，这些都不能说，抱歉。这就是我们正在做的，也是分阶段做的事情。但是我只能告诉大家，在不同的业务要用不同的节奏，相同的业务核心是跟组、敢不敢组的问题。它是一个非常困难的事情。

**问**：为什么一定要AI驱动？那之前的惯性会导致什么？

何小鹏：实际上一直是在软件的逻辑里做一个更复杂的缝合怪。就像你修一个房子，你会有更多的材料、更多的工艺用AI的，但它还是修一个老房子的方法论，修出来的还是一个原来的房子，只是可能修得更快一点。我们想要的是不一样的东西。有很多不一样的东西，最基础的时候是大家要看到它才相信。但作为企业的CEO，我觉得有时你要相信，过程中部分节点能让大家看到，因为可以提高信心、提高领导力。有很多时候是因为你相信去做，不是因为它已经成功。作为一家小型公司和一家大型公司是不一样的。小鹏汽车虽然是一家创业公司，但从人员规模角度已经不算小了（数万人）。所以你如何把人分节奏去做，我觉得有非常不一样的行为。我很难在物理AI上对规划组织来做分享——第一我们在尝试、在Testing，第二任何的规划跟目标都不会对内外分享。为什么有很多事情不需要讨论？因为很难有一个人准确地可以跟你讨论。你只能去听很多人的想法、意见，他们可能大部分是错的，因为他们都不是从全局角度给你的建议，都是从某个角度。全局是自己把握的。过个几十年是不同的朋友和自己的思考逻辑，你要把它汇聚起来，最后你还下赌注，就像创业一样。

**问**：我听说你在过去三年变得更敢赌了？

何小鹏：那也不是。我觉得只是有很多时候知道当你要下注的时候，早点下会更好，但并不是更敢赌了。2022年底小鹏碰到很多挑战的时候，我给了自己两个想法，跟赌都有关系：第一个就是 **绝不服输**；第二个就是 **愿赌服输**。如果你把这两条都能比较好地放在一起去平衡它，意味着即使面对巨大的困难，你都要坚持——也许你再努一把力它就过了。但是你要把心态做得非常好。就像刚才我说的，当我把第二代VOAE放大，并且最后坚定地把第一代VOAE完全停止它，从业务到技术甚至到组织，那是一个巨大的心理上和物质上的双重压力。你越犹豫、越等待、越想观察、越想再说过六个月，你可能越难成功。所以这个里面的度量的判断真的非常有意思。但是我相信再过几年，等我们更好的成功，也许再来奖。

### 关于转型的普遍性问题

**问**：这是今天大家面临的一个普遍问题，对吗？我们一定得转型。

何小鹏：我不完全认同这句话。今天数字AI在哪些方面可以得到好的转型，哪些方面不用。对一家大企业（1万或5万人），**转型晚三天，可能成功概率提高了15%**。对一家微型公司，它今天用筒子化的方法论、筒子化的组织，它没有变化，它可以去赌，因为它小。而且今天的AI能够去帮助我们的，无外乎三种：第一种是简单的分析师（很多地方我都不用分析师了，AI在帮我分析）；第二种是简单的程序员；第三种是简单的设计师。实际上它还是辅助为主，提高了效率和效能，它并没有取代，更没有超大规模的协同效应，它根本还没到达。所以对于这三个方向的小微型公司，适合度更高，速度应该更快；对于跟这三个方向不相关的，可能晚一点。而对于物理AI，今天大家根本都没有想清楚物理AI世界是什么岗位。刚才我讲的三个岗位是哪些最容易、最优先在什么地方落地。绝大部分我看到讲颠覆这个世界、用AI来提高效能的，根本没有概念。所以我自己认为，从某些角度它是对的，从某些角度它是完全没有逻辑的。我们上次觉得物理世界有很多的岗位都会变化，远超过数字世界。因为在这个世界上原来如果说360行，有36000个不同的岗，数字世界的岗是相对有限的。很多人会不是这样的判断——很多人说白领不是比蓝领的岗位更多吗？白领也有很多是在跟数字世界和物理世界交融的岗位。实际上，今天我们如果看物理世界，假设我先说 Google 下注自动驾驶是2009年，到了现在17年。我觉得 Waymo 是一个既好又不好的解决方案——好是技术能力不错，但它很难全球化。在过去的时间里，Level 4技术天然就是一个更高级的缝合怪，它很难在AI里做到极其高强度的泛化。它不是一个极度聪明的方法论。当然 Google 现在在新的大模型里做了很多事情，我认为将来可能会变化，但目前我并没有看到。所以即使是让司机更轻松、更安全、更快捷地开车（我们每个人都是司机），就光这个事情做了17年，没有一家公司能做得极好。换个角度，很多很小型的公司（几条枪、十几个人）说要用AI Coding去改变，我觉得他们完全低估了物理世界改变的多向性和复合性。就像我们当初从移动互联网冲到汽车行业，都觉得把一件事做到极致、再极致一点就更好了——No，因为你考虑的维度不对。它是一个物理的维度，你是一个数字的维度。在数字维度做到极致，它的价值点可能极小。

### 内部反对声音与“用脚投票”

**问**：你在去年下注关键赌注的时候，公司内部有负面的声音吗？有反对的声音吗？

何小鹏：那肯定的。我觉得大部分非AI的主管觉得你不管做A还是B可能都错了，因为那时他们对AI的认知不够强烈。大部分跟AI相关的主管，有些人觉得Yes，有些人觉得No，大部分属于中间态——不确认它是不是一个好的节奏，不确认这是不是能够做到一个落地的效果。实际上连那个时候我都是一样的。但这就是创业的乐趣。每个人的创业不就是在你当时的能力上，10倍到1000倍的可能性吗？如果你要赌1000倍，对不起，你的概率是千分之一；如果是10倍，你的成功概率可能十分之一。所以就像创业最初始的赌注一样，你的创业规模变大，你一样需要去赌，只是说那时你不应该赌1000倍的概率，你可能是赌数倍到10倍之间。

**问**：你听过最大的反对声是什么样的？

何小鹏：我觉得最大的反对声音就是有很多人 **用脚投票**。怎么讲？用脚投票的意思非常简单：他们不相信这个事情，觉得做不到，他就离开了，去做其他事情。在今天，我觉得在一个企业里面，对于组织的思考，切忌不要小刀砍大树、慢慢砍。想清楚了，砍掉它。所以在某些上面你就会敢下注，从组织到流程到方向，全部改。

**问**：所以去年动刀动得很大，对吗？

何小鹏：对。大家所看到的很多都是最上层的业务层或者应用层，但是内部看到的就是一直到根上面去。所以我一直还是想说，上次是如何利用AI驱动某些业务，是非常之重要的，而不是利用硬件去驱动整体的体系。

## 第四部分：机器人——从四足到人形，从不相信到相信

### 机器人的发展三阶段

**问**：你们机器人去年在国内外都挺火的，这个产品是怎么诞生的？我听说你们是在2023年 All in 了通用人形机器人的方向？

何小鹏：小鹏的机器人分三个阶段。第一个是2018到2020年，它是一个独立的团队，跟中国的其他机器团队一样（中国当时有四五家都在做四足）。2020年到2023年是第二个阶段，我们花了差不多三年半到四年，做了三个不同的 Milestone。在这个阶段我们试过很多：用机器人的方法做机器人、用汽车的方法做机器人、中间做了一些缝合的方式做机器人，都取得了不同的成功跟失败。到了2023年之后又是一个新的逻辑。实际上我们在2022年看到了新的模型，觉得机器人原来不可能成功（没有大佬成功的可能性），因为小脑的复杂度都远高过大家。今天很多人说在机器人的小脑已经做好了，我说哪家小脑用同一个单调的步伐往前慢慢地行走？我觉得那不是小脑，那是脊柱或者你的老杆，只是保持平衡，你还远远连小脑都不是。他们说那也叫小脑？实际上只是不同统计逻辑。

### 2023年的转型：从不相信到相信

**问**：23年发生了什么让你从不相信变相信了？

何小鹏：23年是因为当时如果我们用一个对机器人非常懂、非常熟悉的团队，没有人能做出一个好的、全新一代的机器人。当时用了LC（李想？或其他负责人）。所以我们当时把300人的团队只留了不到60人，解散了。他们出去后跟我说有十个创业团队，大部分都拿到钱了。我自已认为要去重构整个机器人的逻辑，不能用以前的纯汽车人，也不能用以前的纯机器人，但是也不能啥都不懂。那应该用什么样的人？我不知道。所以我当时选了另外一个全新的团队，它既懂点AI，又懂点汽车，又懂点工程，又懂点机器人，然后尝试去做一个全新思考的机器人的逻辑。

**问**：为什么当时选中它？

何小鹏：我觉得有很多时候是命运。你不能说因为它既不是专门做机器人的也不是专门做智能驾驶的，所以我选它？它真都不是。我当时觉得它的象限跟我的思考的象限比较吻合。所以它是我在象限内思考选择的几个人之一。首先你选择它，它也选择你。它的反向选择，我认为可以Try。

### 为什么一定要通用人形？

在23年，为什么一定要是通用人形？我觉得很开心我跟我团队LC都一样。LC比我还要Crazy，他一直跟我说他想是 **造人**，而不是造机器人。绝大部分人把一个机器人当做一个商业的产品，但我认为他没有思考机器人最终在这个社会、在这个旅途的参与感，或者跟我们每个人的情绪价值的连接度。就像2014年，很多人觉得智能汽车的智能是没有价值的。站在今天也有很多人不相信通用机器人会到来，也不愿意看到通用机器人也许在20年、50年以后会跟人类的各个领域强相关。今天数字AI只是辅助人类可能三个为主的岗位（还有数十个分支岗位），但物理AI可能是它的数十倍到数百倍的规模。每个人都会变老。我对机器人里面一个很重要的点是：机器人对老人，将来是绝对最重要的两个因素之一——第一个因素是这个能体AI如何让老人不生病、更长寿、更开心；第二个是物理AI上机器人对老人的影响度会非常大，老人很有可能把机器人作为他唯一的依赖。我觉得极少人会这样思考。小鹏的机器人的思考道路跟很多公司是不一样的。

### 去年机器人的爆发性讨论与“委屈”

**问**：去年那波爆发性的讨论有出乎你的意料吗？你当时还挺委屈的。

何小鹏：对，我们是没有想到。我们实际上的想法非常技术化。因为去年的那块机器人是我们前年一季度开始做的，对我们来说它只是我们的一个 **中间版本**。这个中间版本我们只是想认证，如果一个机器人有这么多人那么多的关节（人有200多个关节，还有肌肉和皮肤。我可以告诉大家，今天我们看到的很多机器人，如果它上有肌肉和皮肤之后，它的行走效果会极大程度变坏——这是完全不一样的事情）。我们只是想去探索一下，如果这样的情况下它是不是能够更容易地进入到我们的生活或工作情绪中间。

我举一个例子：以前做了四足机器人（狗、马），不管是四足还是什么，它进到你的房间都不行。如果你让一个大概长度1米1到1米2左右的机器狗进到你的房间，你的房间很快出很多问题。比如它在床头柜那里无法原地掉头。如果你家里有一只金毛，它在调头时尾巴会往上动一下，会抬起，会刮到墙和你的床，你不会觉得它会受伤，也不会觉得床受伤。但是一只机器狗，它100%会让你们两个都觉得受伤。那做小一点呢？如果做得非常小，它的能力级就非常差，只是轻度陪伴，续航特别短。再说一个，如果是一个双足的机器人，全身有了盔甲，很威武，1米8。今天你即使跟它一起（你是它的设计者），你也不愿意跟它只隔了1米行走——有压迫感。你觉得它可能危险、可能很烫、可能有电、可能很脏。反正各种原因你会跟它保持距离感。如果连成年人都这样，那老人和小朋友会怎么看？它在社会上的安全和法律法规该怎么解决？在工业版的机器人可以做成这样的逻辑，因为它本身就不是在家庭或商业里去。所以很多人在工业版机器人里说“我避开了ABC的问题”，但工业版机器人上引发了一个新问题：手部能力要很强。它会引发这样一个新问题。我还是回过头来说：在未来的世界，机器人如果走到人类社会，在很多岗位对基础蓝领和白领都会产生价值，它一定要走到我们身边。所以我们选了一条很难的道。

今天中国的很多机器人公司，无论在质量安全（包括最近的运动会可以看到，就光一个乐高例就会出现这么有趣的现象），还是在它如何进入到真正的物理AI的思考逻辑。我觉得可能我们在两三年前都已经看到了，并且在做调整。但是我们选择了一条很痛苦的道路，这条道路你看了越深，你越难实现。

**问**：为什么不做一些中间态？

何小鹏：我觉得我们已经在做中间态了。我们的中间态只是说，如果你想把机器人真正可以简单地泛化带入到人间，我们已经是在选择最容易做的中间态了。

**问**：这人应该多高？

何小鹏：你看我们现在这一代的机器人，大概在1米69到1米70之间。它要让男生和女生都比较舒适的身高，而且它可以穿衣服、甚至有头发，但是它不能有自己的脸。它必须跟人要有一定的差距。为什么不能有自己的脸？有一些是**恐怖谷效应**，也有一些是其他的法律和社会学的效应。所以实际上现在大家所看到的小鹏的机器人，和小鹏正在做的机器人还是有蛮大的差距。我觉得都等到今年的下半年吧。我期待我们在一个很深度的思考、做的一个最简单的中间态，然后用全公司之力、全小鹏的自力能够做好。

### 发布会上的纠结：要不要证明不是真人？

## 第五部分：技术剧变下的CEO焦虑与心路

### 焦虑源于“公理”的重构

**问**：在这样一个技术剧烈变革的时代，你焦虑吗？作为一个CEO。

何小鹏：当然焦虑。

**问**：有多焦虑？

何小鹏：因为你会发现你以前的很多逻辑论，甚至你逻辑论的底层你心目中的定理、范式，甚至再往底层可能甚至到公理层——就是你的价值观、人生观、世界观，你都会觉得在这些东西的变化下，好像以前的这一套逻辑不是那么 Work。但是它该怎么去 Work？**看到问题、解决问题、和构建体系（既有上限又能堵住下限、避免问题）是三种完全不同的能力级**。很多人只看到问题。我当年没创业的时候，我也看到无数问题。当你作为一个创业者，你发现有很多问题天然应该存在。你根本没法解决问题——你是个CEO，一样解决不了。你是一个更大的有权力的人，你都解决不了。因为你看到的问题是世界的某个角度的一环而已。所以这是一个非常有意思的话题。所以你刚才说的焦虑就是，你会不断地重构你的定理甚至公理。

**问**：推翻过过去什么样的公理？有什么过去觉得非常坚定相信的，今天觉得在这个时代下不 Work？

何小鹏：我举一个跟这个不相关但有一点关联的例子：当时小鹏做了自动驾驶很多年之后，你越做下去越觉得 Level 5 永远不会到达。因为你觉得用软件在全球所有的场景、法规、人机交互（人机工程）你会发现是无限循环的。但是当你真正用AI重构这个事情之后，你觉得它可能存在了。当它可能存在的时候，你就会对很多原来的所谓壁垒和逻辑产生新的想法。我只能点到这里了。
