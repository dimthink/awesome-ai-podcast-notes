### 向量世界的度量衡：范数（Norm）与距离（Distance） `[01:15-03:10]`

**核心观点**

在深入探讨正交性之前，我们需要掌握向量的基本度量概念：范数（即向量的长度）和向量间的距离。

**深度阐述**

李老师指出，7.1章节的内容相对“无聊”，因为它主要集中在各种名词的定义，但这些基础概念却是构建后续复杂理论的基石。他相信其中一些概念，如向量的长度，同学们可能在高中阶段就有所耳闻。

首先引入的是**范数（Norm）**。范数本质上就是向量的“长度”。在课本中，一个向量 `v` 的范数通常用双竖线 `||v||` 来表示。如果未作特殊说明，我们通常指的是欧几里得范数（L2范数），它的计算方式是将向量 `v` 中每个元素的平方和再开根号。

**重要原话**：“Norm的意思其实就是一个 vector v 的长度，就是它的 norm。” `[01:40]`

例如，对于向量 `v = [1, 2, 3]`，其范数 `||v||` 的计算过程是：`sqrt(1^2 + 2^2 + 3^2) = sqrt(1 + 4 + 9) = sqrt(14)`。

紧接着，李老师定义了**两个向量之间的距离（Distance）**。如果有一个向量 `u` 和另一个向量 `v`，它们之间的距离被定义为这两个向量相减后所得新向量的范数。换句话说，`distance(u, v) = ||u - v||`。

他通过一个具体的例子来巩固理解：假设向量 `u = [2, -3, 0]` 和 `v = [1, 2, 3]`。要计算它们之间的距离，首先计算 `v - u = [1-2, 2-(-3), 3-0] = [-1, 5, 3]`。然后，计算这个差向量的范数：`||[-1, 5, 3]|| = sqrt((-1)^2 + 5^2 + 3^2) = sqrt(1 + 25 + 9) = sqrt(35)`。这个 `sqrt(35)` 就是向量 `u` 和 `v` 之间的距离。

李老师通过这些直观的例子，清晰地阐明了范数和距离的计算方法，为后续更复杂的概念打下了坚实的基础。他强调这些都是非常直接和简单的概念，旨在帮助学生建立对向量空间几何直观的理解。

**精华收获**

范数是向量的长度，通常指欧几里得范数，计算为元素平方和的平方根。两向量间的距离是它们差向量的范数。

### 正交性（Orthogonality）的引入：高维空间的“垂直” `[03:10-06:00]`

**核心观点**

正交性是高维空间中“垂直”概念的推广，其数学定义是两个向量的点积为零。

**深度阐述**

在定义了范数和距离之后，李老师正式引入了本讲的核心概念——**正交性（Orthogonality）**。在讲解这个词之前，他特意花了一点时间，以一种非常生动有趣的方式，尝试教大家如何正确发音。他播放了YouTube上的发音教学视频，并模仿发音，甚至开玩笑地用“台湾国语”来比喻发音，例如“Ortho就是台湾国语这样。Orthogonal。就是你杀了一个东西这样。” `[04:00]` 这种轻松幽默的互动，瞬间拉近了与学生的距离，也让这个略显生僻的专业词汇变得更加亲切和易记。

在发音环节之后，李老师回到了数学定义。他首先回顾了**点积（Dot Product）**的概念，指出这同样是高中阶段就学过的知识。对于两个向量 `u = [u1, u2, ..., un]` 和 `v = [v1, v2, ..., vn]`，它们的点积 `u · v` 定义为对应元素相乘再求和：`u · v = u1`*`v1 + u2`*`v2 + ... + un`*`vn`**。*

**重要原话**：“今天如果两个 vector u 跟 v，它们的 inner product 是零，那它们就是 Orthogonal。” `[04:40]`

这就是正交性的核心定义：如果两个向量 `u` 和 `v` 的点积（或称内积，inner product）为零，那么它们就是正交的。

李老师随即解释了正交性与我们日常生活中“垂直”概念的关系。他强调，在二维和三维空间中，两个向量垂直，它们的点积就等于零，这是我们中学时就学过的几何事实。而**正交性，正是将这种“垂直”的概念推广到更高维度的空间**。在高维空间中，我们很难直观地“想象”出什么是垂直，什么是角度，因此，通过点积为零的代数条件来定义“垂直”关系，就变得尤为重要和实用。

**重要原话**：“这不就是垂直吗？没错，其实 Orthogonal 就是二维跟三维空间中的垂直。只是在高维空间中垂直这件事，因为在高维空间中你就很难想像说什么叫做垂直、什么叫做角度这样。所以，在高维空间中的垂直就叫做 Orthogonal。” `[05:00]`

最后，李老师补充了一个直观的推论：**零向量（zero vector）与任何向量都是正交的**。这是因为零向量的任何元素都是零，它与任何向量进行点积运算，结果都必然是零。这个性质进一步巩固了对正交性定义的理解。

**个人感受**

李老师在讲解“Orthogonal”发音时的幽默感和亲和力，让枯燥的数学概念变得生动有趣。他将高维抽象概念与低维直观理解相连接，极大地降低了学习门槛。

**精华收获**

正交性是高维空间中“垂直”的数学表达，定义为两向量点积为零。零向量与任何向量正交。

### 范数的通用定义：p-范数 `[06:10-07:50]`

**核心观点**

范数有一个更通用的定义，即p-范数，它允许我们计算不同“类型”的向量长度，并强调在奇数p值下取绝对值的重要性。

**深度阐述**

在助教讲解完作业后，李老师继续深入讲解范数的概念，引入了其更具一般性的定义——**p-范数（p-norm）**。他指出，之前介绍的欧几里得范数（L2范数）只是p-范数的一种特殊情况。

**通用p-范数的表示方法**是 `||v||`*`p`**，其定义为：将向量 **`v`** 中每个元素 **`vi`** 取绝对值后，进行 **`p`** 次方运算，然后将所有结果相加，最后再开 **`p`** 次方（即取 **`1/p`** 次幂）。数学表达式为：**`||v||`*`p = (sum(|vi|^p))^(1/p)`。

李老师特别强调了在计算中**取绝对值 ****`|vi|`**** 的重要性**。

**重要原话**：“in general 而言就是把 v 里面的每一个 element vi 统统都取绝对值以后乘上 p 倍，然后它再统统加起来，再开 p 次方或取它的 p 分之1 次方。” `[06:40]`

他解释说，如果 `p` 是偶数（例如 `p=2`），那么 `vi^p` 总是非负的，此时取不取绝对值结果都一样。但如果 `p` 是奇数（例如 `p=1` 或 `p=3`），并且 `vi` 为负数，那么 `vi^p` 就会是负数。在这种情况下，如果不取绝对值，最终的求和结果可能会导致开 `p` 次方时出现问题（例如，负数开奇数次根号虽然有实数解，但为了保持范数作为“长度”的非负性，通常会先取绝对值）。因此，为了确保范数始终代表一个非负的长度，通用定义中要求先对每个元素取绝对值。

他举例说明了不同 `p` 值下的范数：

- `p=1` (L1范数)：`||v||`*`1 = sum(|vi|)`**，即向量元素绝对值之和。*
- `p=2` (L2范数/欧几里得范数)：`||v||`*`2 = sqrt(sum(vi^2))`**，即我们之前介绍的长度。*
- `p=3` (L3范数)：`||v||`*`3 = (sum(|vi|^3))^(1/3)`**。*
- `p=4` (L4范数)：`||v||`*`4 = (sum(|vi|^4))^(1/4)`**。*
通过引入p-范数，李老师扩展了学生对向量“长度”概念的理解，使其能够适应更广泛的数学和应用场景。

**精华收获**

p-范数是范数的通用形式，通过 `(sum(|vi|^p))^(1/p)` 定义。在计算时，对向量元素取绝对值至关重要，尤其当 `p` 为奇数时，以确保范数代表非负长度。

### 范数平方展开的实例演示 `[11:40-14:10]`

**核心观点**

通过一个具体的例子，演示如何利用点积和范数的性质，展开一个形如 `||2u + 3v||^2` 的表达式。

**深度阐述**

为了巩固前面所学的点积和范数性质，李老师在黑板上详细演示了如何展开一个复合表达式 `||2u + 3v||^2`。这个例子是理解这些性质实际应用的关键。

**视觉/结构信息描述**：李老师在黑板上逐步写下推导过程，从范数平方到点积，再到点积的分配和结合，最后回到范数平方。

1. **范数平方转换为点积**：
根据性质 `||X||^2 = X · X`，我们将 `||2u + 3v||^2` 转换为 `(2u + 3v) · (2u + 3v)`。

**重要原话**：“以后你看到 norm 的平方就可以把它拆成那个 dot product。” `[12:30]`

1. **点积的分配律**：
将 `(2u + 3v) · (2u + 3v)` 展开，就像代数乘法一样，利用点积的分配律：

`= (2u) · (2u + 3v) + (3v) · (2u + 3v)`

`= (2u) · (2u) + (2u) · (3v) + (3v) · (2u) + (3v) · (3v)`

1. **点积与标量乘法结合**：
利用 `(cu) · (dv) = cd(u · v)` 的性质：

`= 4(u · u) + 6(u · v) + 6(v · u) + 9(v · v)`

1. **点积的交换律**：
由于 `u · v = v · u`，中间两项可以合并：

`= 4(u · u) + 12(u · v) + 9(v · v)`

1. **点积转换回范数平方**：
再次利用 `X · X = ||X||^2` 的性质：

`= 4||u||^2 + 12(u · v) + 9||v||^2`

通过这个详细的推导过程，李老师不仅展示了如何一步步应用之前介绍的性质，也加深了学生对这些性质之间相互转换的理解。这个例子是理解向量代数运算灵活性的一个绝佳示范。

**精华收获**

通过将范数平方转换为点积，并结合点积的分配律、标量乘法结合律和交换律，可以系统地展开复杂的向量表达式，最终将结果表示为范数平方和点积的组合。

### 毕氏定理（Pythagorean Theorem）：正交向量的长度关系 `[14:10-16:50]`

**核心观点**

毕氏定理（勾股定理）在线性代数中被推广到正交向量，即如果两个向量正交，则它们和向量的范数平方等于各自范数平方之和。

**深度阐述**

李老师接着引入了一个大家耳熟能详的定理——**毕氏定理（Pythagorean Theorem）**。他以幽默的方式开场，询问同学们是否听过这个定理，然后揭示其真面目：“其实你是听过的，你只不知道它英文而已。这个是毕氏定理啊，就是他国小就学过的东西。” `[14:40]` 这种接地气的表达，瞬间消除了学生对英文术语的陌生感，将抽象的数学概念与日常所学联系起来。他甚至还对“Pythagorean”这个发音和“毕达哥拉斯”的中文译名进行了有趣的探讨。

**定理内容**：如果向量 `u` 和 `v` 是正交的（Orthogonal），那么它们和向量 `u + v` 的范数平方等于 `u` 的范数平方加上 `v` 的范数平方。

即：如果 `u · v = 0`，则 `||u + v||^2 = ||u||^2 + ||v||^2`。

李老师解释说，这正是我们小学中学时学过的直角三角形两边平方和等于斜边平方的推广。在二维平面上，如果 `u` 和 `v` 垂直，那么 `u + v` 构成直角三角形的斜边，这个定理就完美对应了。在高维空间中，虽然我们无法直观地画出“垂直”，但“正交”的概念依然保持了这种长度关系。

**定理证明**：

证明过程非常简洁，直接利用了前面学过的范数平方展开和正交性定义：

1. 从 `||u + v||^2` 开始展开。根据范数平方转换为点积的性质，它等于 `(u + v) · (u + v)`。
1. 利用点积的分配律，展开为 `u · u + u · v + v · u + v · v`。
1. 根据点积的交换律 `u · v = v · u`，合并中间项：`u · u + 2(u · v) + v · v`。
1. 再将点积转换回范数平方：`||u||^2 + 2(u · v) + ||v||^2`。
1. **关键一步**：由于 `u` 和 `v` 是正交的，它们的点积 `u · v` 等于零。
1. 因此，`||u + v||^2 = ||u||^2 + 0 + ||v||^2 = ||u||^2 + ||v||^2`。
这个证明清晰地展示了正交性如何简化了向量长度的计算，并将其与我们熟悉的几何直觉联系起来。

**个人感受**

李老师将“毕氏定理”与“勾股定理”联系起来，并用“国小就学过”来形容，这种亲切的表达方式，让学生在学习新知识的同时，感受到知识的连贯性和熟悉感。

**精华收获**

毕氏定理在线性代数中指出，若两向量正交，则它们和向量的范数平方等于各自范数平方之和。其证明直接利用了范数与点积的转换以及正交性定义。

### 平行四边形定理：对角线正交的菱形 `[16:50-19:00]`

**核心观点**

如果一个平行四边形的两条对角线是正交的，那么这个平行四边形一定是一个菱形。

**深度阐述**

李老师接着提出了另一个与几何图形相关的定理：如果一个平行四边形的两条对角线是正交的，那么它必然是一个菱形。他首先解释了“菱形”的定义，即四条边长度都相等的四边形，或者说，由两个等长向量构成的平行四边形。

**视觉/结构信息描述**：李老师在黑板上画了一个平行四边形，其相邻的两条边分别由向量 `u` 和 `v` 表示。

1. **表示对角线**：
1. **正交条件**：
定理的条件是这两条对角线是正交的，这意味着它们的点积为零：`(u + v) · (u - v) = 0`。

1. **定理证明**：
`= u · u - u · v + v · u - v · v`

结论是，如果平行四边形的两条对角线正交，那么构成其相邻边的两个向量 `u` 和 `v` 的长度必须相等。这正是菱形的定义，因此这个平行四边形一定是一个菱形。这个定理再次展示了正交性在几何性质推导中的强大作用。

**精华收获**

若平行四边形的两对角线正交，则其相邻边向量长度相等，从而证明该平行四边形为菱形。证明依赖于对角线向量表示和点积展开。

### 三角形不等式：向量长度的几何直觉 `[19:00-20:50]`

**核心观点**

三角形不等式指出，任意两个向量之和的范数，小于或等于它们各自范数的和，这与几何中“两边之和大于第三边”的直觉相符。

**深度阐述**

李老师接下来讲解的是另一个基础而重要的定理——**三角形不等式（Triangle Inequality）**。他再次将其与我们熟悉的几何概念联系起来：“我们知道说，呃，这个两边，的，三角形的两边的长度之和一定大于等于第三边。” `[19:00]`

**定理内容**：对于任意两个向量 `u` 和 `v`，它们之和的范数 `||u + v||` 总是小于或等于它们各自范数的和 `||u|| + ||v||`。

即：`||u + v|| <= ||u|| + ||v||`。

**定理证明思路**：

为了证明这个不等式，通常会从平方形式入手，因为范数平方可以转换为点积，便于代数运算。

1. 从 `||u + v||^2` 开始展开：
`||u + v||^2 = (u + v) · (u + v) = u · u + 2(u · v) + v · v = ||u||^2 + 2(u · v) + ||v||^2`。

1. **关键一步**：引入点积的性质 `u · v <= |u · v|`。
由于点积 `u · v` 可以是正数、负数或零，而 `|u · v|` 总是非负的，所以 `u · v` 总是小于或等于其绝对值。

因此，我们可以得到一个上界：

`||u||^2 + 2(u · v) + ||v||^2 <= ||u||^2 + 2|u · v| + ||v||^2`。

1. **核心工具：柯西-施瓦茨不等式（Cauchy-Schwarz Inequality）**。
李老师指出，下一步需要用到柯西-施瓦茨不等式，即 `|u · v| <= ||u|| ||v||`。

将柯西-施瓦茨不等式代入上一步的表达式：

`||u||^2 + 2|u · v| + ||v||^2 <= ||u||^2 + 2||u|| ||v|| + ||v||^2`。

1. **识别完全平方项**：
右侧的表达式 `||u||^2 + 2||u|| ||v|| + ||v||^2` 正好是一个完全平方项，等于 `(||u|| + ||v||)^2`。

1. **最终推导**：
所以我们得到 `||u + v||^2 <= (||u|| + ||v||)^2`。

由于范数都是非负的，对两边同时开平方根，即可得到最终的三角形不等式：

`||u + v|| <= ||u|| + ||v||`。

李老师强调，在这个证明过程中，唯一需要额外证明的就是柯西-施瓦茨不等式。这表明柯西-施瓦茨不等式是三角形不等式成立的基石。

**精华收获**

三角形不等式 `||u + v|| <= ||u|| + ||v||` 表明向量和的长度不大于各向量长度之和。其证明关键在于利用 `u · v <= |u · v|` 和柯西-施瓦茨不等式 `|u · v| <= ||u|| ||v||` 将范数平方展开式转化为完全平方形式。

### 柯西-施瓦茨不等式（Cauchy-Schwarz Inequality）的证明 `[20:50-25:30]`

**核心观点**

柯西-施瓦茨不等式 `|u · v| <= ||u|| ||v||` 是线性代数中的一个基本不等式，可以通过向量归一化和非负性性质进行证明。

**深度阐述**

李老师指出，要完整证明三角形不等式，我们必须先证明其核心支撑——**柯西-施瓦茨不等式**。这个不等式在数学中具有极其广泛的应用。

**定理内容**：对于任意两个向量 `u` 和 `v`，它们的点积的绝对值 `|u · v|` 总是小于或等于它们各自范数的乘积 `||u|| ||v||`。

即：`|u · v| <= ||u|| ||v||`。

李老师提供了两种证明思路：一种是直观的几何解释，另一种是更严格的代数证明。

**1. 直观的几何想法（通过归一化向量）**：

1. **归一化向量**：首先，将向量 `u` 和 `v` 归一化，即把它们变成长度为1的单位向量。
**重要原话**：“如果你把这个 u 呢，把它除掉它的 norm，你把 u 呢除掉它的长度，它就会变成一个长度是1的 vector。” `[21:20]`

1. **单位向量的点积**：两个单位向量 `w` 和 `z` 的点积 `w · z`，其值必然介于 -1 和 1 之间。
**重要原话**：“如果我们今天把两个长度是1的 vector 做 dot product，它的值呢，一定会小于等于1到 负1之间。” `[22:00]`

这是因为点积 `w · z = ||w|| ||z|| cos(theta) = 1 `*` 1 `*` cos(theta) = cos(theta)`。而 `cos(theta)` 的取值范围就是 `[-1, 1]`。

1. **推导柯西-施瓦茨**：
因此，`|w · z| <= 1`。

将 `w` 和 `z` 替换回 `u` 和 `v` 的形式：`|(u / ||u||) · (v / ||v||)| <= 1`。

根据点积与标量乘法的性质，可以将范数项提取出来：`|u · v| / (||u|| ||v||) <= 1`。

最后，将 `||u|| ||v||` 乘到右边（由于范数非负，不等号方向不变）：`|u · v| <= ||u|| ||v||`。

李老师称这种方法为“直觉的想法”，因为它依赖于对单位向量点积的几何直觉。

**2. 严格的代数证明（利用范数的非负性）**：

这种方法不依赖于几何直觉，而是纯粹利用点积和范数的代数性质。

1. **定义单位向量**：同样，我们定义 `w = u / ||u||` 和 `z = v / ||v||`。
1. **利用范数的非负性**：任何向量的范数平方都必须大于或等于零。考虑向量 `w + z` 的范数平方：
`||w + z||^2 >= 0`。

1. **展开范数平方**：
`||w + z||^2 = w · w + 2(w · z) + z · z`

由于 `w` 和 `z` 是单位向量，`w · w = ||w||^2 = 1` 且 `z · z = ||z||^2 = 1`。

所以，`1 + 2(w · z) + 1 >= 0`。

简化后得到 `2(w · z) + 2 >= 0`，即 `2(w · z) >= -2`，从而 `w · z >= -1`。

1. **证明另一侧不等式**：为了证明 `w · z <= 1`，我们需要考虑另一个向量 `w - z` 的范数平方：
`||w - z||^2 >= 0`。

展开：`||w - z||^2 = w · w - 2(w · z) + z · z`

同样，`w · w = 1` 且 `z · z = 1`。

所以，`1 - 2(w · z) + 1 >= 0`。

简化后得到 `2 - 2(w · z) >= 0`，即 `2 >= 2(w · z)`，从而 `w · z <= 1`。

1. **结合结果**：将 `w · z >= -1` 和 `w · z <= 1` 结合起来，得到 `|w · z| <= 1`。
1. **替换回原始向量**：最后，将 `w` 和 `z` 替换回 `u` 和 `v` 的形式，即可得到柯西-施瓦茨不等式 `|u · v| <= ||u|| ||v||`。
李老师在课堂上快速地提到了“把正号改成负号”来得到另一侧不等式，这里我们为了完整性，明确地展示了 `||w - z||^2` 的推导。这个严格的证明，完全依赖于向量代数的基本性质，不依赖于任何几何直觉，因此在数学上更加严谨。

**个人感受**

李老师在证明柯西-施瓦茨不等式时，先给出直观解释，再尝试严格证明，这种教学方式非常有效。他坦诚地指出直觉的局限性，并引导学生思考更严谨的证明方法，体现了科学的求真精神。

**精华收获**

柯西-施瓦茨不等式 `|u · v| <= ||u|| ||v||` 可通过将向量归一化为单位向量 `w, z`，然后利用 `||w + z||^2 >= 0` 和 `||w - z||^2 >= 0` 展开并简化，最终推导出 `|w · z| <= 1`，从而证明原不等式。
