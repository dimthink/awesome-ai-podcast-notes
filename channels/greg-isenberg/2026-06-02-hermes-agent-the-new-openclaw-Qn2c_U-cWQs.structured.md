# Hermes Agent 完全指南：安装、配置与高效使用

本指南根据 **Greg Isenberg** 与 **Imran** 的深度对话整理，全面介绍了 **Hermes Agent（赫尔墨斯代理）** 的安装、配置、技能开发以及在 Android 设备上的运行方法。无论你是刚接触个人 AI 代理，还是希望从 **OpenClaw** 迁移，本文都将提供清晰、完整的步骤与实用建议。

---

## 为什么选择 Hermes Agent？

Imran 在尝试使用 **OpenClaw** 时遇到了三个主要问题，这也是他最终转向 **Hermes Agent** 的原因：

1. **缺乏内置记忆系统**：每次都需要重复告诉代理做同样的事情，没有记忆能力。
1. **网关频繁崩溃**：有时甚至需要每小时重启一次，大量时间浪费在设置上而非实际使用。
1. **Token 消耗不透明**：无法看到 Token 的去向和消耗原因，成本难以控制。
作为对比，**Hermes Agent** 提供了以下核心优势：

- **内置记忆系统**：每次成功完成任务后，代理会自动写入自身记忆，使用标准 SQLite 数据库，可实时检索历史成功记录。即使忘记保存 API 密钥等环境变量，它也能通过搜索历史日志帮你找回。
- **更高的稳定性**：Imran 表示使用超过一周无需重启，远优于 OpenClaw。
- **成本可见且可控**：通过 **OpenRouter** 等路由服务，可以清楚看到每个模型的费率，从而选择性价比最高的方案。
此外，**Hermes Agent** 开箱即用，内置 **40 多种工具**（包括浏览器、网页搜索、定时任务、图像生成、家庭助手集成等），并预装了诸多热门技能，例如在 MacBook 上可直接使用 Apple Notes、Apple Reminders、Find My、iMessage 等，无需手动添加。

## 安全性如何？

对于用户关心的安全问题，Imran 提供了以下方法：

- **主动审计**：你可以直接让代理检查你的安全设置，例如询问“我的配置安全吗？”它将扫描你的计算机，指出是否暴露了密钥、是否存在明文密码、防火墙配置是否薄弱等。
- **隔离运行**：Hermes Agent 支持在 **Docker 容器** 内运行，与主机文件隔离；也可以部署在 **Modal** 上作为无服务器服务，提供更灵活的隔离选项。
- **日常维护**：Imran 个人选择在裸机上运行，但坚持每天更新并定期要求代理加固配置。

## 安装 Hermes Agent

**适用平台**：macOS（需安装 Xcode 开发者工具）、Linux、Windows Subsystem for Linux（WSL）

**安装步骤**：

1. 访问 **Hermes Agent 官方文档**（位于 New Research 网站）。
1. 复制并执行一行安装命令。如果是首次在 Mac 上安装此类工具，需要先运行：
`bash`

xcode-select --install

1. 安装完成后，在终端输入 `Hermes` 即可启动。你可以跳过首次引导，直接关闭。
1. 如需更新，重新执行安装命令即可。
**最重要的命令**：`Hermes model` — 用于查看和管理可用的模型提供商。

---

## 模型选择与成本控制

在终端输入 `Hermes model`，你将看到所有支持的提供商列表。Imran 推荐使用 **OpenRouter** 或 **Nebula Portal** 来大幅降低 Token 费用。

- **OpenRouter** 提供了多种模型及实时价格，有时还会有免费模型（例如 Nvidia 的 Neumotron 曾免费开放）。通过清晰的价格列表，你可以选择性价比最优的模型，例如：
- **节省技巧**：对于每天都在执行的重复任务，可以一次性编写代码（例如使用免费模型生成），之后直接运行代码，无需每次调用 LLM，从而大幅节约 Token。这样能将 Token 开支降低 **90% 以上**（Imran 的经验：从每 5 天约 $130 降至每 5 天约 $10）。

## 集成与远程访问

### Telegram 集成

Hermes Agent 原生支持 **Telegram** 连接，你可以创建多个代理（Imran 以“布偶角色”命名），通过 Telegram 与它们对话，就像在终端中一样。

### Tailscale 配置

推荐安装并配置 **Tailscale**，将你的手机和所有计算机置于同一个虚拟网络中。这样你可以通过任何终端 SSH 应用远程访问代理，监控并与它交互。

### 与 Obsidian 集成

**Obsidian（一款知识管理工具）** 是 Imran 极力推荐集成的应用。Hermes Agent 可以自动生成 Markdown 笔记，并组织成每日摘要，包含：

- 本周重要事项
- 今日待办
- 即将到来的旅行
- 工作和个人事务
Imran 表示，如果没有代理，他绝不会每天花精力手动整理 Obsidian，但代理替他完成了这一切，让原本枯燥的笔记管理变得轻松而有趣。要获得类似的效果，建议 **连续使用 Hermes Agent 7 天以上**，让代理充分了解你的习惯，然后它会为你创建结构化的 Obsidian 工作流。

### G Stack 集成

**G Stack（由 Y Combinator CEO Garry Tan 开发的工具集）** 旨在将 Y Combinator 风格的创业方法论引入代理：每周迭代、提出正确的问题、改进产品并付诸实施。这份原本只属于 YC 内部的知识，现在已作为免费技能集成到 Hermes Agent 中。如果你在创业，强烈建议安装 G Stack 技能，帮助优化产品决策。

## 在 Android 手机上安装 Hermes Agent

Imran 专门展示了一个运行在 **Solana Seeker Android 15 手机** 上的 Hermes 实例，通过 **Termux**（Android 终端模拟器）安装。具体步骤：

1. 安装 **Termux** 应用。
1. 安装 **Termux API**（通过 F-Droid 开源应用商店下载），该 API 可访问手机传感器数据。
1. 使用电脑端相同的安装脚本，在 Termux 内完成部署。
安装后，代理可以访问 Android 手机的各项功能：

- 电池信息
- 更改 Wi-Fi 网络
- 调节音量
- **拍摄照片**
- 调节屏幕亮度
- 触发振动马达
- 读取短信（可用于自动化两因素认证）
- 甚至模拟屏幕点击
**为什么用 Android 手机而非 Mac Mini？** 因为 Android 手机成本低、功耗低、可随身携带，还可插入 SIM 卡，成为一款始终在线、低功耗的专用代理设备。

**赚钱思路**：利用 Termux API，你可以实现设备原生的社交媒体自动化（直接从手机发布内容，避免第三方调度工具导致的限流）、大规模运营账号、甚至自动化处理短信验证码等。

## 高效使用 Hermes Agent：关键提示语

为了让代理更好地服务你，Imran 推荐以下提示语（Prompt）：

- “我一直在拖延什么事情？”（代理可读取你的待办清单）
- “今天最应该专注的事情是什么？”
- “我每天重复做的哪些任务可以或应该自动化？”
- **“今晚你可以为我构建什么工具，让明天的生活更轻松？”**
- “今天有什么重要的事情被我忽略了？”
这些提示语能帮助你发现自动化机会，将重复性工作交给代理。

## 设置多个代理：工作 vs 个人

Imran 建议至少设置两个代理：

- **个人代理**：处理日常琐事、健身、菜谱、个人财务等。
- **工作代理**：专注于工作任务，如邮件分类、开发者工具、与同事协作等。
这样既保持数据隔离（尤其在工作电脑上），也让代理各司其职。你可以为每个代理指定不同的模型，简单任务用廉价模型，复杂任务用更强大的模型。但 Imran 认为，对于大多数用户，**一个或两个代理是最优选择**，以避免过度管理。

---

## 构建你自己的技能

Hermes Agent 的强大之处在于可自定义技能。Imran 分享了几个例子：

- **个人财务技能**：自动分析银行对账单。
- **健身与膳食技能**：Imran 曾用本地语音转文字模型录制了 8 分钟的语音，描述冰箱和储藏室的所有食材，然后让代理每天根据这些食材和健身目标推荐三道菜。
- **经典心理治疗技能**：将 MIT 教授 Joseph Weizenbaum 花费三年构建的 ELIZA 聊天机器人（一个自然语言处理治疗程序）的 Wikipedia 描述作为技能加载，代理即可扮演类似的自我成长顾问。
这些技能只需提供相关文件或描述，代理就能将其转化为可交互的功能。

---

## 注意事项：保持更新与避免过度定制

- **每天更新**：Hermes Agent 仍处于测试阶段，建议每日更新以获得稳定性和新功能。Imran 曾有一个实例 9 天未更新，落后了 535 分钟。
- **锁定配置**：通过 Telegram 或 WhatsApp 远程访问时，请注意安全。
- **定制不是目的**：Imran 强调，**学会定制并不是真正的技能**，利用代理完成实际任务、节省时间、提升产出才是关键。不要陷入不断优化的陷阱，让工具为你服务，而不是相反。

## 结语

Imran 总结了他使用 Hermes Agent 的最大收获：“在一家基金工作时，我能与更多创始人交流并开展更高质量的对话，因为大量背景工作由代理处理。如果因此多接触 20% 或 30% 的公司，我们将获得更好的信号、更优质的交易流，最终基金回报也更丰厚。这让我在工作中更出色。”

如果你希望拥有一个学习你习惯、自动执行重复任务、全天候运行的个人代理，**Hermes Agent** 是一个值得深入探索的选择。
