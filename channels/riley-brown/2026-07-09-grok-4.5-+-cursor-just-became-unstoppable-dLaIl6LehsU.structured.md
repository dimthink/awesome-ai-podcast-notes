# Grok 4.5 + Cursor：已成为无可阻挡的力量

## Grok 4.5 模型简介

Cursor 现在是 SpaceX 的一部分，他们刚刚发布了一款全新的模型——**Grok 4.5**。这个模型性价比极高，性能与 **Opus 4.8** 相当，但速度更快、价格更便宜。当你在 Cursor 中使用这个模型，尤其是配合全新的 **Canvas**（画布）功能时，使用体验变得更加有趣。下面我将为你演示。

## Canvas 初体验：创建着陆页

在 Cursor 中，你可以看到一个新的可用模型：“Cursor Grok 4.5 High Fast”。我输入提示：

> “请为 Grok 4.5 创建一个着陆页，将此模型与其他前沿模型公司的产品进行比较，并放在一个 Canvas 中，包含实用的图形以及人们对这个模型的评价引述。”

Cursor 响应非常快，很快就生成了一个整洁的着陆页。我们可以全屏打开，关闭侧边栏，并继续调整。例如，我可以说：

> “请将这个页面居中，现在它是左对齐的。请居中。”

然后我再次使用设计模式（design mode），选中图表并说：

> “我喜欢这个图表。能再做一些类似的图表吗？用这些颜色，我很喜欢。在上面加一个，让它有用。”

模型迅速在 Cursor 内部生成了研究报表。我非常喜欢这个全屏窗口，这也是我使用新 Cursor Grok 4.5 模型的主要方式。

## 价格比较与性能评价

我制作了一个 artifact（工件），将输入和输出 token 的价格合并为一个价格。**Fable 5** 是每百万 token 输入加输出 60 美元。**Grok 4.5** 仅为每百万 token 输入加输出 8 美元。与 **Opus 4.8** 相比（许多人认为 Grok 4.5 的性能与 Opus 4.8 非常接近），Grok 4.5 便宜了近四倍。

正如 Elon Musk 所说：

> “我们内部的评估是，Grok 4.5 大致与 Opus 4.7 相当，但速度更快。能力、更快的速度和更低的成本的结合使其具有竞争力。我们正在接近实际有用性，而不是基准测试。但我们可以自己判断。”

## 重制个人网站

接下来，我按下 Command+N 创建一个新聊天。我新建了一个工作区（workspace），放在一个普通文件夹中。我输入：

> “嘿，请查看 rileybrown.xyz，那是我个人网站。我想让它焕然一新，做得非常好。我想参考 Cursor 的着陆页作为示例，但不要太像 Cursor，只是那种颜色调色板。我想创建一个明显更强、在移动端和网页端都优化得更好的新着陆页。”

模型开始工作。与此同时，我决定创建一个移动应用。

## 构建 iOS 语音应用

我创建了一个新工作区，打开一个新文件夹“mobile app testing Grok”，然后要求 Cursor：

> “创建一个移动应用，使用 Swift，设计非常酷，黑色背景，允许我通过 Grok 的语音代理进行聊天。这里是文档链接：…… 使用这个 API 创建一个高质量的 Swift 应用。我希望能在模拟器中运行并测试。”

我粘贴了 API 密钥。模型飞快地完成了任务，只用了约 3 分 30 秒。我可以在模拟器中运行了。测试对话：

> Grok: “你好。很高兴见到你。我是 Grok，你温暖而机智的伙伴。”

> 我: “你能听到我吗？”

> Grok: “我不能直接听到你。我是基于文本的，但你可以通过这个聊天输入或说话，我会倾听。”

> “你能修复一下吗？我的音频有问题，我无法中断 AI 模型，声音断断续续。请仔细思考并修复这个问题。”

> Grok: “我仍然在这里倾听你的每一个字……”

虽然有一些小问题，比如消息发送了两次，但应用仍然成功创建，我能够与 Grok 聊天，这很酷。我还在 iPhone 上启动了它，测试了对话，Grok 都能正确回应。

## 完善网站与设计迭代

现在回头看看我几分钟前创建的着陆页。我发现模型直接将其推送到了我的实际个人网站。我要求它改为本地运行再进行修改。

进入全屏模式，关闭左侧面板，我可以用设计模式继续调整。我快速检查：

- 这个有点太大了。
- 不需要“add tag”。
- 使其更可见，使用更偏紫的颜色。
- 点击页脚： “请改善页脚，用最好的判断，为个人网站添加页脚，可能包含网站上其他链接的引用，以及社交链接再次出现。”

模型迅速生成，效果不错。我还要求把“views”改为视图图标，并显示“242K”而不是具体数字。就这样我们不断向 Grok 发送提示，它每次都快速做出改变。每个更改只需 10 秒左右。

## 创建 Excalidraw 克隆

为了进一步测试新模型，我打开一个新文件夹“Excalidraw”，要求：

> “请创建一个完美的 Excalidraw 克隆，但我想能够将图形变成演示文稿。这是一个 Web 应用，所有数据需要完美存储。创建一个数据库，不需要认证。这是我个人网站的一部分，但数据要完美存储。”

模型开始工作，它选择了 Convex（一个后端平台）并直接部署到 Vercel（一个部署平台）。大约 5 分钟后就部署完成了。我在浏览器中打开实际链接，可以正常使用：

- 新建绘图，输入文字“Hello there. My name is Riley.”，改变颜色等。
- 可以使用箭头工具等。
- 页面显示“Created by Grok”和一个部署链接。

这是一个实际运行在互联网上的 Excalidraw 克隆。

## 总结与观察

现在我们回顾一下做了什么：

1. **个人网站着陆页**：重新设计并部署到了 Vercel，设计不错。
2. **语音移动应用**：可以用语音与 Grok 实时对话。
3. **Excalidraw 克隆**：功能完整的绘图工具，立即部署。

我的主要观察是：
- **速度非常快**。
- **价格便宜**：比 Opus 便宜四倍。
- **使用乐趣**：配合 Cursor 的设计模式，可以快速迭代。
- **Canvas 功能**：可以创建工件并共享给团队。
