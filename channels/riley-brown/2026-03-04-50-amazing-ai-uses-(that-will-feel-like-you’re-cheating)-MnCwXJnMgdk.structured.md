# 50 个让你事半功倍的 AI 创意应用案例

你是否知道，AI 可以在业务、内容创作甚至日常工作中，为你提供如此多的帮助？在本视频中，我将分享 50 个我每周都会使用的创意 AI 应用案例。我是 Riley Brown，在旧金山经营一家初创公司，年营收远超过七位数。同时，我在各大社交平台拥有 1500 万粉丝，而 AI 是我取得这些成绩的重要原因。

我们将一起探讨语音模型、图像模型、视频模型，以及音乐和音效的生成。如果你们喜欢这类视频，我每周都会制作多次关于创意使用 AI 的内容。所以，请务必点击订阅和点赞。话不多说，让我们直接开始吧。

## 一、智能对话与文本生成

### 1. 与 AI 直接聊天

我们要谈到的第一个 AI 应用案例，我相信你一定听说过，那就是与 AI 聊天。我可以对它说：“**快速给我一个可以制作视频的点子。**” AI 真的能与我对话，并帮助我解决几乎所有我能想到的问题。稍等一下，它现在已经给了我一大堆不同的想法。

### 2. 利用记忆功能进行个性化聊天

另一个能让与 AI 聊天体验更佳的功能是使用**记忆功能**。我可以指示 AI：“**利用我们过去对话的记忆，专门为我提出一些个性化的想法。**” 现在，AI 会实际运用记忆，查看最近的聊天记录。如你所见，它确实浏览了我之前所有的聊天记录，并想出了一些点子。这些想法都是根据我们之前的对话量身定制的，这非常实用。

*顺便一提，我使用的是 Claude，而不是 ChatGPT，但 ChatGPT 也能实现类似功能。这两个工具非常相似，我个人认为 Claude 更适合高级用户，这也是我喜欢它的原因。*

### 3. 结合网络搜索与 AI 聊天

你还可以让 AI 在联网搜索的帮助下进行对话。你会注意到，如果你点击这里的“工具”按钮，可以开启一个“研究”功能，这实际上是**深度研究**，与普通的网络搜索不同。当然，你也可以进行基础网络搜索。例如，我可以让 AI：“**搜索互联网，看看 Matt Wolf 最近的视频和他谈论的内容，然后为我 brainstorm 一些我也擅长制作的视频创意。**”

现在，它理论上会执行多项任务：使用记忆功能，同时搜索互联网。你可以看到它正在使用“工具”进行网络搜索，经过思考和推理后给我答案。如你所见，它在搜索了整个互联网后给出了回复。它不仅仅去了 YouTube，还访问了各种不同的网站，查看了 Matt Wolf 的讨论内容，并进行了一些额外的思考。你可以看到它的思考过程，然后我们得到了这个答案。

### 4. 让 AI 模仿你的特定写作风格

你还可以与 AI 聊天，并让它按照特定的风格进行写作。在 Claude 中，你可以通过“搜索和工具”找到“风格”功能，尝试创建新的自定义风格并添加写作示例。但我认为 Claude 的这个功能不太好用，更好的方法是创建一个**项目**。

我新建一个名为“**片头风格写作**”的项目。在项目说明中，我会粘贴我之前视频的片头脚本，并指示：“**这是一个我以前做的视频片头。当被要求撰写片头时，请使用这种风格，但不要使用其中的任何主题内容。**”

然后，我让它“**写一个关于 Perplexity 20 个应用案例的视频片头，按照说明中的风格来写。**” 并使用 Opus 4.5 模型。生成的片头是：“我使用 Perplexity 已经有一段时间了，它无疑是世界上最好的 AI 搜索引擎。但只有当你真正知道如何正确使用它时，它才能发挥最大功效。这是世界上最强大的研究工具，你可以用它来替代 Google、ChatGPT，甚至是昂贵的研究助理。你可以用它来工作、学习，等等。” 这个效果相当不错。如你所见，这与它参考的原始片头风格非常相似。因此，这是让 AI 模仿你风格的一个非常好的方法，远比使用内置的风格功能要好。我认为创建项目的方式要好得多。

### 5. 语音转文字

第五个 AI 应用案例是**语音转文字**，这个功能非常棒。我可以使用一款叫做 Whisperflow 的工具进行语音输入，比如我说：“**好的，这太棒了。请写下一段。**” 它就能将语音转化为文字。Whisperflow 有非常慷慨的免费计划。我现在几乎不怎么用键盘了，除了制作像这样的内容时，我基本上都是在对着电脑说话。

### 6. 语音转文字 + 图表绘制

将语音转文字与图表工具结合使用，效果更佳。例如，在使用 Excalidraw 时，配合语音和键盘快捷键，你可以非常轻松地添加箭头、复制元素等。这在制作视频、为社交媒体创建图表等教育用途上非常有用。这是一种很有趣的体验，我强烈推荐。这个图表工具叫 **Excalidraw**，如果你看过我的视频，就会知道我在几乎每个视频里都会用到它。它是我最喜欢的工具，默认字体很好看，而且一旦你掌握了各种快捷键，使用起来就非常方便。

### 7. 文字转图表

另一个应用案例是在 Excalidraw 中使用**文字转图表**功能。我可以复制“Perplexity 的 20 个应用案例”这段文字，然后在 Excalidraw 中选择“文字转图表”，粘贴提示词，点击生成。它就会实际创建一个图表，你可以直接将其插入到你的画板中。

你还可以指定图表的样式，例如，我说“**制作一个简单的思维导图**”，然后点击生成。它就会创建一个更简洁的思维导图，你可以直接插入应用。你可以继续在图表上书写，这是一种通过 Excalidraw 快速获得可视化的很酷的方法。

## 二、图像与视频创作

### 8. 在终端中创建高质量游戏和应用

接下来，我们要进入一些更高级的内容，我们将使用 Warp 终端中的 **Gemini 3**。打开 Warp 并注册后，你可以使用 Gemini 3 Pro。我们可以输入任何想法，例如：“**用 3JS 创建一个看起来像宝可梦的有趣 2D 游戏。**” 然后按回车。

Warp 的酷炫之处在于，它可以同时创建多个东西。我们可以同时让它创建一个“**用 3JS 制作的水循环趣味模拟**”和一个“**名为‘Riley’s Shop’的披萨店的有趣着陆页**”。我会再次选择 Gemini 3 Pro，因为它最擅长处理视觉化的事物，如着陆页、模拟和游戏。

所有这些都在 Warp 中同时创建。我们可以运行命令，并在它们之间切换。Warp 给了你很多控制权，让你能够引导 AI 创建应用。不用担心这看起来像个终端，其实没那么难。我们允许所有操作，然后就可以在浏览器中运行它们，真正玩游戏、看模拟和着陆页。

- **宝可梦风格游戏**: 服务器在 localhost 801 运行，点击进入，就得到了这个小巧的 3D 游戏。虽然没创建角色，但你可以在这个单一房间里移动，撞到墙壁会停下来。
- **水循环模拟**: 在 localhost 802 运行。哇，看这个！我们可以看到蒸发，甚至可以调整灵敏度，这太疯狂了。
- **披萨店着陆页**: 在 localhost 803 运行。这就是用 Warp 和 Gemini 3 创建的 Riley's Shop 的酷炫着陆页。
这只是用 AI 创建的三种不同类型的应用。如果你想尝试，可以去 warp.dev 下载 Warp。

### 9. 用 AI 生成图像

秉承 Gemini 3 是最具视觉化模型的理念，谷歌发布了另一个非常视觉化的模型，一个可以生成图像的模型。这正是我现在要向你们展示的。如果我们进入 gemini.google.com，可以选择这个“创建图像”功能。输入“**橡皮鸭**”，它就会生成一张鸭子的图片。

### 10. 用 AI 编辑图像

我们还可以用 AI **编辑图像**。我们可以对刚才生成的鸭子图片说：“**请把鸭子放在一个由巧克力糖浆构成的湖上，并加入漂浮的棉花糖。**” 它就会实际编辑那张刚创建的图片。完成后，可以看到它精确地保留了鸭子本身，只是把它放在了巧克力糖浆上并添加了棉花糖。

### 11. 替换 YouTube 缩略图中的人物

你可以替换任何 YouTube **缩略图中的人物**。找一个 YouTube 视频链接，通过在线工具获取其缩略图并复制下来。回到 Gemini，粘贴图片，然后说：“**好的，我想你把这张缩略图里的家伙换成这个人。**” 接着，去 Twitter 复制你的个人资料图片，也粘贴到 Gemini 中，并指示：“**保持一切不变，只把这个人换掉。**”

这就是使用**参考图像**生成新图像。你可以同时使用多达 14 张参考图片。AI 会识别这些图片，并将你放到那个 YouTube 缩略图中。

### 12. 使用更专业的工具（Crea）进行图像编辑

不过，这并不是我使用图像编辑最偏爱的方式。有一个更好的工具叫 **Crea**，它是我最喜欢的 AI 图像工具，因为它可以访问所有模型，包括视频模型。在 crea.ai/app 上，我进入编辑区域，上传原始缩略图。切换到 **Nano Banana Pro** 模型，这允许我们使用参考图像。上传之前的 Twitter 头像，然后指示：“**请把这个家伙换成这个人，别让他的手放脸上，给他一件橙色衬衫。**”

使用 Crea 的编辑功能，它会保持原始图像的纵横比，我们只希望替换人物而不改变尺寸。生成四个选项后，它完美地将我的头像添加到了缩略图上，效果非常好。

### 13. 用 AI 编辑图像中的文字

选中最佳的一张，我们可以接着**编辑图像中的文字**。输入指令：“**除了文字，什么都不要改。把文字改成‘The Best’，背景用黄色；再加上‘Vibe Coder’，文字用白色。**” 点击生成，它就完成了。缩略图上的文字就从原来的变成了“The Best Vibe Coder”，我们有四个选项可以微调文字。

### 14. 用 AI 编辑图像中的物体

我们还可以用 AI **编辑图像中的物体**。再次使用 Nano Banana，选中我们最喜欢的那张图，然后指示：“**把这三个手机换成别的东西。我想让你把它们换成一张电影感的图片，图片里是一台电脑，显示着一个非常高质量的着陆页。让它大一点，就在‘Vibe Coder’的正下方，箭头保留，占据缩略图左下角的大部分区域，成为这个缩略图的视觉中心。除了去掉手机并添加一个大电脑，其他什么都别改。**”

Nano Banana Pro 的厉害之处在于，它底层有一个推理模型，你可以用非常自然的语言与它交流，几乎不需要考虑如何写提示词。点击生成，它就完成了从原图到新图的转变。效果非常惊人。

### 15. 用 AI 编辑图像中的光照

我们甚至可以用 AI **编辑光照**。我们可以说：“**请在人物和电脑显示器后面添加一个不太强烈的白色发光阴影。让它看起来有电影感，并针对 YouTube 缩略图进行优化。**” 你几乎不需要再知道如何手动遮罩，只需用语言提示，就像在 AI 模型中直接内置了 Photoshop 一样，这真的太不可思议了。完成后，效果再次令人惊叹。

### 16. 使用“修复”功能进行精准区域编辑

第 19 个（根据原文顺序）AI 应用是精准区域编辑。使用 Crea 的**画笔工具**，我们可以选择图像中想要改变的具体部分。选中衣服区域，然后说：“**把他的衬衫改成棕色，做成高领衫，上面写着‘Riley’。**” 生成四个编辑结果，AI 只会编辑被遮罩的区域。它完成了从原图到新图的转变，改变了高领衫并添加了文字。

### 17. 实时图像生成

在 Crea 上，还有一个我觉得很有趣但实用案例不多的是**实时图像生成**。例如，提示词是“三个棒球飞过空中，像素艺术”，左侧的图像会实时变化。我们可以修改提示词，比如“三个粉红色的棒球飞向一个巨大的游泳池”，图像会实时更新，加入池子和水花。如果把“棒球”改成“飞艇”，图像里的物体就会变成飞艇。再改成“粉红色的雪人”，就会变成雪人飞过空中。这个功能更多是为了娱乐，如果能保持一致性，可能会非常有趣。

### 18. 用 AI 增强图像

找到理想的缩略图后，我们可以**用 AI 增强图像**。下载图片后，在 Crea 的“增强”功能里上传。我更喜欢使用 **Topaz** 模型，认为它效果很好。可以将图片放大 2 倍（对于 YouTube 缩略图来说，16 倍太多了），目的是让图像更清晰锐利。点击增强，还可以调整锐度。

使用 Topaz Generative 模型，并加上“**使皮肤光滑**”的提示。对比增强前后的图片，原来的皮肤有点奇怪，处理后皮肤光滑多了，整体质感更好，文字也更清晰了。

### 19. 为首帧生成图像

我们继续使用 Crea。为了制作视频，我们可以先**为首帧生成图像**。使用 Nano Banana Pro，上传一张联合创始人的照片作为图像提示，并加上提示词：“**一个男人在外面草地上，穿着白色背心和蓝色牛仔裤。**” 将比例设为 3:2，然后生成多张图像。

### 20. 从首帧生成视频

有了首帧图像，下一步就是**从首帧生成视频**。下载一张满意的图片，进入 Crea 的视频标签，选择 **VO 3.1**（谷歌的模型，我认为是世界上最好的视频模型）。上传这张图片作为起始帧，并加上提示词：“**男人坐在外面放松，背景播放着轻柔的音乐。镜头拉近。他说：‘嘿，我叫 An，欢迎来到我的世界。’**” 然后生成视频。视频生成需要比图像更长的时间。

### 21. 从首帧和尾帧生成视频

如果我们想要对整个视频有更多控制，可以先生成一个**尾帧图像**，然后**从首帧和尾帧生成视频**。我们可以用 Nano Banana Pro，基于首帧生成尾帧：“**除了一切不变，让男人挥手，镜头稍微拉近一点，在他右侧远处的背景中有一个人在跑步。**” 生成一系列尾帧，选择满意的下载。

回到视频标签，上传之前那张图片作为**起始帧**，再上传新生成的这张作为**结束帧**。加上提示词：“**漂亮的拉近镜头，轻柔地播放自然的声音。男人挥手。**” 再次用 VO 3.1 生成视频。由于使用了结束帧，视频结尾应该能看到有人在背景中开始跑步。

## 三、音频处理与生成

### 24. 自定义系统提示词

（此处接回原文关于 prompt engineering 的部分）

普通的 Claude 桌面应用或网页版聊天体验是为广泛人群优化的。而 platform.claude.com（控制台）则提供了原始模型，你可以创建自己的**系统提示词**。例如，你可以设定：“**你将扮演一名记者。**” Claude 本身很擅长处理系统提示词，你可以先用普通 Claude 帮你构思一个好的提示词，比如一个“**第一性原理思考者**”的提示词。

在控制台粘贴这个系统提示词，然后输入用户问题：“**向我解释一下什么是经济。**” 运行后，AI 就会按照设定的格式给出答案。你可以不断调整系统提示词，直到满意为止，例如让它在回答开头加上一句特定的问候语。

### 25. 将系统提示词用于应用开发

得到满意的系统提示词后，你可以在自己的应用中重复使用它。你可以从控制台获取相应的代码，稍后交给 AI 模型，让它创建一个能与这个特定 AI 角色聊天的应用。

### 26. 文本转语音

接下来谈谈声音生成。首先是**文本转语音**。在 11 Labs 上，输入文字：“**你好啊朋友们。我是 Riley Brown，我准备好找点乐子了。**” 你可以选择或创建自己的声音。我之前已经训练了一个自己的声音（只需上传约一分钟的音频即可）。选择我的声音，生成语音，听起来就非常像我。

### 27. 声音转换

另一个是**声音转换**。上传一个音频文件，比如我录制的“你好朋友们，我是 Riley”。然后选择将声音转换成别人的，比如我的联合创始人 An 的声音。生成后，我的语音就会变成 An 的音色，虽然因为说话方式和口音不同，听起来不完全像他说话，但音色确实更接近 An 了。

### 28. 生成音效

你还可以用 AI **生成音效**。输入提示词：“**海边多风的一天**。” 调整提示词影响度，可以生成音效。如果觉得时长太短，可以指定时长，比如 8 秒，并细化提示词：“**夏日海边放松的声音，背景有几个人在交谈。**” 生成的音效效果相当不错，可以轻松下载保存。

### 29. 人声隔离

**人声隔离**功能。如果你有一个包含很多背景噪音的音频文件，想要只提取人声，就可以使用这个工具。它会分析整个音轨，然后生成一个只包含人声的新版本，效果非常好。我发现这对视频编辑非常有用。

### 30. 音乐生成（11 Labs）

在 11 Labs 上还可以**生成音乐**。进入音乐标签，输入提示词：“**酷炫的嘻哈节奏，一个男人在说唱 AI 模型（如扩散模型和 LLM）有多厉害。**” 这是第一个公开提供 API 的 AI 音乐模型，可以集成到你自己的应用中。生成的音乐示例确实不错。

### 31. 高质量音乐生成（Suno）

不过，11 Labs 的音乐质量明显不如 **Suno**。Suno 是世界上最好的音乐模型。我们可以生成歌曲，比如：“**一首关于 AI 模型变得太强大并将取代人类工作的阴郁合唱。**” Suno 会生成多首歌曲，旋律和背景音乐都非常出色。

### 32. 生成电影配乐/纯音乐

在 Suno 上，我们还可以选择**生成纯音乐**，比如电影配乐。选择“纯音乐”选项，输入提示词：“**紧张感上升的管风琴强烈音乐，电影配乐，电影中的精彩场景。**” 这对于视频编辑非常有用，我经常在制作短视频时使用这些纯音乐作为背景。

## 四、信息整理与演示文稿

### 33. 生成研究报告

假设我想了解 Suno 是如何被训练的，我可以去 Claude 让它生成一份**研究报告**。使用之前那个“第一性原理思考者”的提示词，让它解释这些模型是如何被训练和创造的。

### 34. 用 AI 创建演示文稿

得到研究报告后，我们可以用 AI 来**创建演示文稿**。这里我们使用 **Gamma**，这是我现在非常喜欢的一个工具。将 Claude 生成的研究报告复制到 gamma.app，选择“从笔记或大纲生成”。Gamma 会将其浓缩成一份可展示的文稿。你可以选择主题，例如选择一个喜欢的主题，它会生成包含多张卡片的演示文稿。

你还可以让 Gamma 使用 AI 生成图像，选择 **Nano Banana Pro** 模型，并加入“**图像使用极简信息图风格**”等额外关键词。Gamma 会同时生成文稿和 AI 图像。你甚至可以编辑生成的图片，调整提示词，选择最合适的一张放入幻灯片中。这是一个非常简单快捷的创建幻灯片的方法，而且可以随时更改主题。

## 五、综合应用：“氛围编码”构建应用

### 40. 创建带付费墙的应用

最后 10 个案例，我们将进入“**氛围编码**”的部分。在我看来，它能将所有可能性融合在一起。“氛围编码”很像提示工程，但它是可视化的，可以说是**视觉化的提示工程**，你可以用它来创建软件体验。

第一个例子是一个**语音笔记应用**，我们给它加上了**付费墙**。这意味着你可以实际向用户收费，这非常酷。在 Vibe Code 这类应用中，你只需要进入支付标签，按下一个按钮，按照指引设置即可。这是一个可以发布到应用商店的完整移动应用。你只需复制提示词，比如“**创建一个每月 9.99 美元的订阅**”，然后就可以实际发布到应用商店了。借助 AI，你可以构建应用，然后将其提交到应用商店。

### 41. 从系统提示词构建应用

还记得我们之前在 Claude 控制台创建的那个“第一性原理思想家”的**系统提示词**吗？我们可以用它来创建一个全新的应用。在 Vibe Code 应用中，创建一个新应用，命名为“**Prince Chat**”，并指示：“**这是一个用第一性原理解释主题的聊天应用。先不要添加聊天功能，我们下一步再加。**” 使用 Claude Opus 4.5 模型，并启用后端以存储所有相关数据。应用正在被创建，它在手机和网页上是实时同步的。

### 42. 为应用集成 API

在等待应用创建的同时，我们可以设置下一个功能。我想用 **Grok 4**，认为它更擅长第一性原理思考。我可以指示 AI：“**请使用这个 API（指 Grok 4）来创建我聊天的这个角色。不要用 Anthropic，用 Grok。这是系统提示词。**” 将之前的系统提示词粘贴进去。这样，在应用前端生成后，它会接着设置使用 Grok 4 API 的后端。因此，我们可以为移动应用构建前端、后端，并添加 API 集成。

### 43. 查看应用数据库

应用创建完成后，可以在手机上看到 **Prince Chat**。我们试一下：“**解释一下披萨是如何制作的。**” Prince 开始思考并给出了答案。很有意思的是，我们可以通过“**数据库查看器**”看到所有应用数据。比如用户的提问和助手的回答，以及创建时间等。我们实际上已经创建了一个数据库。

### 44. 为应用重新设计 Logo 和界面

接下来，我们可以**重新设计应用**。在 Vibe Code 的图像标签中，确保“图标”模式开启，输入提示词：“**3D 渲染的紫色皇冠，带有白色边框和金色皇冠，卡通图标。**” 生成一系列图标，选择喜欢的。然后向 AI 发出指令：“**请把这个作为左上角的 Logo。同时把界面做得更好一点，让输入消息框的背景不是紫色而是灰色，像短信那样。并在 AI 回复的头像旁边使用这个 Logo。**” AI 会很快对应用进行修改。

### 45. 为应用添加音效

我们还可以为应用**添加音效**。生成一个“**轻微的加载提示音**”，选择一个喜欢的。然后指示 AI：“**当 AI 开始思考加载时，以 40% 的音量播放这个音效。**” 应用会随之更新。

### 46. 为应用添加触感反馈

同样，我们可以为应用**添加触感反馈**。在手机上效果更佳。我们可以选择一个“**柔和的触感**”，然后指示 AI：“**当我按下回车发送消息时，使用这个触感反馈。**” 这样，每次发送消息时手机都会有一个轻微的震动。

### 47. 使用“选择”模式进行精准编辑

Vibe Code 还提供了“**选择模式**”进行精准编辑。在手机上切换到选择模式，可以选取界面中的特定元素，比如顶部的图标。选择后，可以查看该元素的代码，并发出修改指令：“**请不要用这个图标，换成三个点的图标，点击后在一个模态框中打开选项。‘删除’应该是一个选项。请再添加两个按钮选项，但先不实现它们的功能。**” 应用会相应地进行修改。

### 48. 通过截图和圈画进行 Debug

如果在修改后出现了 BUG（例如弹出的菜单被聊天框遮挡），我们可以进行**截图并圈画出问题**。保存截图后，在 Vibe Code 的图像标签中上传这张截图作为参考图像。然后发出指令：“**请使用另一种模态框。这个不好用，被聊天框挡住了。把它做成居中的弹出框。**” 应用会理解你的意图并尝试修复。

### 49. 修改应用功能（如清空聊天）

在应用里点击“清空聊天”后，发现它真的删除了数据库里的所有消息。如果你不希望这样，可以进一步指示 AI 修改这个行为，比如改成软删除或二次确认。

### 50. 将应用发布到应用商店

应用构建完成后，你只需要将其连接到你的苹果开发者账户，就可以直接**发布到 App Store**。而且，通过 API 标签页，你可以将今天讨论的很多功能（如 Nano Banana、各种音频视频模型）轻松地通过 API 集成到你的应用中。
