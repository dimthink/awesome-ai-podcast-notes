## 适合谁

- 想系统跟踪 AI 产品、模型、Agent、创业和工程实践的人；
- 想快速判断一集长播客是否值得完整收听的人；
- 想把公开访谈、课程、分享转成可复盘知识库的人；
- 想基于公开内容做检索、主题索引或二次整理的人。

## 三种文稿版本

- **深度笔记**：默认入口，适合阅读、复盘和引用。
- **结构化文稿**：更接近提纲 / 数据结构，适合后续做主题索引、检索或再加工。
- **原始文稿**：尽量保留转写文本，适合核对细节和追溯上下文。

不是每一集都有三种版本；具体以每篇笔记顶部的“版本”区和 `data/episodes.json` 为准。
