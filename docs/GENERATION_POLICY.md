# Generation Policy

Awesome AI Podcast Notes is generated by YouTube AI Deep Reader.

The pipeline converts public long-form AI videos, podcasts, lectures, and conversations into Markdown notes and indexes. The GitHub repository is a public distribution format for the generated archive.

## What The Pipeline Produces

- Main deep notes for reading and review.
- Structured notes when the upstream processing has produced a stable outline.
- Raw transcript-style text when available.
- Public JSON indexes for browsing and reuse.

## Public Privacy Boundary

The public GitHub archive does not expose private Notion links, credentials, internal database IDs, or private workflow URLs.

`notion_url` is kept as an optional schema field for private deployments, but it is intentionally empty here.

## Quality Boundary

AI-generated notes can be useful, but they are not final truth.

Common failure modes include:

- speech recognition mistakes;
- speaker attribution errors;
- missing context from slides, screen shares, or visual demos;
- over-compressed summaries;
- outdated claims from old source material;
- hallucinated connective tissue between source points.

Readers should verify important claims against the linked source.

## Copyright And Attribution

This archive links back to source material whenever a URL is available.

The original media, titles, people, channels, and any quoted source material remain under the rights of their respective creators and platforms. The generated notes are intended for learning, review, search, and discovery, not as a replacement for the original content.
