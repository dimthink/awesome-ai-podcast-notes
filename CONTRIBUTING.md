# Contributing

Thanks for helping improve Awesome AI Podcast Notes.

This repository is primarily generated by an automated pipeline, so the best contributions are usually about curation, metadata, documentation, and quality review.

## Good Contributions

- Recommend an AI podcast, YouTube channel, course, lecture, or long-form interview.
- Fix metadata such as title, channel name, source URL, publish date, language, or duplicate records.
- Report broken links, empty notes, low-quality transcripts, or obviously wrong summaries.
- Improve topic tags, channel grouping, bilingual documentation, or project explanations.
- Propose curated collections for agents, LLM engineering, AI product work, research, startups, or design.

## How To Open An Issue

Please include:

- the source URL;
- what is wrong or missing;
- the suggested correction;
- any context that helps verify the change.

Chinese issues are welcome. 中文 issue 可以直接写中文。

## Pull Requests

Small documentation and metadata fixes are welcome.

For generated episode files, please explain why a manual edit is needed. The daily publisher may overwrite generated files, so durable fixes should ideally be made through the data source or publisher logic.

## Quality Bar

- Keep source links intact.
- Do not add private Notion links or credential-bearing URLs.
- Do not present AI-generated notes as official transcripts.
- Prefer clear factual corrections over broad rewriting.
- Respect the original creators and link back to the source material.
