# Roadmap

This project is moving from an automated archive toward a more useful open knowledge base for long-form AI content.

## Now

- Keep daily publishing stable.
- Preserve three content versions when available: deep note, structured note, and raw transcript.
- Maintain channel and timeline indexes.
- Publish a machine-readable episode index.

## Next

- Build a topic index for agents, LLM engineering, AI product, AI coding, startups, research, and creative tooling.
- Normalize channel metadata and source types.
- Add curated collections for common learning paths.
- Improve generated note quality checks and review labels.
- Add a lightweight public search or GitHub Pages view.

## Later

- RSS or Atom feeds for latest notes and selected topics.
- Better bilingual summaries.
- Quality scores or human-review status.
- Cross-links between related episodes, people, companies, papers, and tools.
- Public contribution templates for channel suggestions and metadata fixes.

## Non-goals

- This is not a replacement for the original podcasts or videos.
- This is not an official transcript archive.
- This project will not publish private Notion links or private workflow data.
